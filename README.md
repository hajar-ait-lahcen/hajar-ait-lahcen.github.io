# Mon CV Interactif - L2 Informatique

Ce projet consiste en la réalisation d'un **CV numérique et interactif** développé dans le cadre du module de Programmation Web. Il présente mon parcours académique, mes compétences techniques ainsi que mes projets réalisés.

## 🚀 Démo en ligne
Le CV est consultable en ligne via GitHub Pages à le lien suivant:
**(https://hajar-ait-lahcen.github.io/)**

## 🛠️ Technologies utilisées
Ce projet intègre différentes technologies web pour offrir une expérience utilisateur dynamique :
* **HTML5 & CSS3** : Structure et mise en forme responsive (Flexbox/Grid).
* **JavaScript / jQuery** : 
    * Animation des barres de compétences au chargement.
    * Système d'accordéon pour le parcours académique.
* **ReactJS** : 
    * Composant réutilisable pour l'affichage des cartes de projets.
    * Gestion dynamique de la section Contact.
* **FontAwesome** : Utilisation d'icônes vectorielles pour les réseaux sociaux et les sections.

## 📁 Contenu du dépôt
* `index.html` : Fichier principal contenant le code source (HTML, CSS, JS/React).
* `profil.png` : Photo d'avatar utilisée pour le profil.
* `menu.mp4` : Vidéo de démonstration intégrée dans la section projets.
